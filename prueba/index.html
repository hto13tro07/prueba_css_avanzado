    • HTML Y CSS para principiantes

    HTML Y CSS son dos herramientas importantes para dar inicio en el mundo del desarrollo web. por lo tanto, en este curso aprenderas todo sobre HTML Y CSS, desde lo mas basico hasta un nivel intermedio, logrando desarrollar tu primera pagina web respnsiva, sin la necesidad de aplicar librerias externas para CSS.

    Instructor: Manuel Gonzalez

    Estilos con SASS

    SASS es una herramienta importante hoy en dia para todo desarrollador web, ya que permite crear etilos mas ordenados, eficientes y con menos lineas de codigo. Por ello, en este curso aprenderas todo lo necesario para crea hjas de estilo desde la sintaxis de SCSS, implementando tecnicas propias de SASS

    Instructor: Manuel Gonzalez
